## Google Calculator & units and currency converter  
http://www.googleguide.com/calculator.html  

`<Super>C` - toggle dialog  
`<Escape>` - close  
`<Ctrl><Up`/`<Ctrl><Down>`/`<Tab>` - navigation  
`Right-click` or `<Enter>` - copy result to the clipboard
